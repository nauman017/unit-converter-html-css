# unit-converter-html-css
This is a simple Unit Converter webpage built using only HTML and CSS as part of my frontend development learning journey. It includes sections for:

🌡️ Temperature – Celsius to Fahrenheit

⚖️ Weight – Kilograms to Pounds

🛣️ Distance – Kilometers to Miles

🧰 Features
✅ Semantic HTML5 structure

✅ Basic layout using CSS

✅ Navigation to scroll to each section

✅ Clean and responsive design

❌ JavaScript logic not yet added – will be implemented in future versions

📚 Purpose
This project is part of my practice while learning HTML, CSS, and JavaScript. My focus is currently on mastering the basics of layout and structure before adding interactivity with JavaScript.
